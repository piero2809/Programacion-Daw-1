#### 1.-Introducción breve y contextualización - 25% de la nota del ejercicio
Una variable es un contenedor de información que almacena un valor en especifico, esta puede ir variando de acuerdo a lo que necesitemos que se guarde en ella. Estas se etiquetan desde un inicio y quedan definidas asi hasta el final, lo unico que puede cambiar es su contenido

Por otro lado, para poder imprimir o mostrar las variables en pantalla se usa el comando "document.write", este sirve como una salida.


#### 2.-Desarrollo detallado y preciso - 25% de la nota del ejercicio
Primero se declaran las variables, estas son "Contenedores" de información. Pueden contener tanto números como letras

Luego se les asigna un valor a cada una de ellas, en este caso a la variable edad se le declara que vale 30, a la variable nombre se le declara "Juan Perez" y a la variable equipo "Juan Perez"

Luego se imprimen el valor de las variables usando "Document.write", este sirve como una salida de codigo en HTML

Este es un ejemplo de como se crean y se definen las variables
					
                                                               ↓
```
var nombre = "Piero"              // Se crea y se define la variable nombre como "Piero"
Var edad = 19                    // Se crea y se define la variable edad como 19
Var deporte = "Basketball"      // Se crea y se define la variable deporte como "Basketball"
```

#### 3.-Aplicación práctica - 25% de la nota del ejercicio

La restricción principal es que solo se pueden utilizar las variables mencionadas y no se permite escribir literales directamente dentro de las sentencias de impresión.*
